# ERP Project — Explainable Clustering (CLASSIX Reproduction)

MSc Data Science, University of Manchester

## Project Overview

This repository contains the Extended Research Project (ERP) for M10, focusing on reproducing and extending the CLASSIX clustering algorithm with explainability analysis.

## Repository Structure

```
├── docs/              # Project documents (proposals, ethics review, PDF)
├── notes/             # Research notes and bilingual project plans
├── src/               # Python reproduction scripts
├── results/           # Experiment results and benchmark metrics
└── official_classix/  # Reference: official CLASSIX library source
```

## Key Files

- `docs/ERP_M10.pdf` — Main ERP report
- `src/reproduce_classix_minimal.py` — Minimal CLASSIX reproduction
- `src/reproduce_official_classix_shape_subset.py` — Shape subset experiments
- `notes/CLASSIX_REPRODUCTION_NOTES.md` — Detailed reproduction notes
